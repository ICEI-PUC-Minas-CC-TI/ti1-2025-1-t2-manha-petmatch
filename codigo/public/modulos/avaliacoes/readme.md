# PetMatch - Página de Avaliações

Esta é a página de avaliações da plataforma PetMatch, desenvolvida como parte de um projeto web. A página permite que os usuários visualizem a média de avaliações da plataforma, a distribuição de notas por estrelas e as resenhas deixadas por outros adotantes.

## Descrição

A interface foi projetada para ser clara, objetiva e acessível. A seção principal mostra o total de avaliações recebidas, a média geral (exibida numericamente e com estrelas) e um gráfico de barras horizontais que representa a quantidade de avaliações por nota (de 1 a 5). Também há uma área com depoimentos reais de usuários, incluindo nome, foto e comentário.


Um botão "Avalie-nos!" está disponível para incentivar o usuário a compartilhar sua própria experiência com a PetMatch.

## Funcionalidades

- Exibição da média geral de avaliações (nota de 0 a 5)
- Gráfico de barras com distribuição de avaliações por nota
- Lista de resenhas de usuários
- Chamada para ação para avaliar a plataforma

## Ferramentas Utilizadas 

HTML, CSS, Estrutura estática (com backend e JavaScript futuramente)
